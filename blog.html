1. What is the Document Object Model(DOM)

Ans: The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

2. How do you select an element from the DOM

Ans: I can select in many ways, by getElementById,getElementsByClassName,getElementsByTagName,querySelector,querySelectorAll
Example: const element = document.getElementById("idName");

3. What is event delegation in the context of the DOM, and why is it useful

Ans: Event delegation is a technique in the DOM where a single event listener is added to a parent element to handle events for its child elements.

It's useful because:- Instead of adding multiple event listeners to many child elements, we add a single listener to the parent, this improve performance. Dynamic event handling. Simpler Code Maintenance.

How do you manipulate an element attributes and styles using the Dom

The style property allows us to set inline styles directly on an element.
Exmaple: element.style.color = "red";

We can set a CSS property using the setProperty method.
Exmaple: element.style.setProperty("margin", "10px");
